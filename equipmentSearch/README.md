#README

##安装与卸载

直接使用 mudlet 的 package manager install/uninstall  功能即可。

##使用方法

安装成功后会自动增加一个 alias：seq <equipment-name>，可以在 mudlet 命令行直接输入，示例如下：

```
seq magic cloak
```

该命令会返回：

```
Searching...
{
  {
    "A magic cloak",
    "fur",
    "831390",
    "2.640kg",
    "great cloak",
    "+5 Spr",
    "Giantkillers"
  }
}
```
